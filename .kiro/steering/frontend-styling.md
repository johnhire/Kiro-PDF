# Frontend Styling Standards
 
## Styling Framework
- Use Massachusetts Design System CSS for all new UI development.
- Do not create custom CSS files unless necessary.
- Prefer utility classes over inline styles.
 
## Component Styling
- Keep styles colocated with components.
- Use responsive utilities.
- Support dark mode.
 
## Design System
- Primary button: background/adaptive/brand/primary-mid
- Border radius: radius MD
- Spacing scale: Massachusetts Design System defaults
